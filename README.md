# HeartRateProject
<br>APP:使用BLE連接穿戴式裝置紀錄心跳數據並上傳至後台
<br>Server:與APP進行連接，接收從心跳數據並儲存至SQL之中
<br>Web:呈現心跳數據與特徵值的圖表
<br>API:撈取SQL資料
